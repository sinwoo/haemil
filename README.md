## GitHub Pages 배포

이 저장소의 디자인 가이드는 GitHub Pages로 퍼블리싱되도록 설정되어 있습니다.

- 퍼블리시 URL (예상) : https://sinwoo.github.io/haemil/
- 메인 페이지는 `index.html`로 리디렉션되며, 실제 가이드 파일은 `designguide.html`입니다.

자동 배포는 `main` 브랜치에 푸시하면 `.github/workflows/pages.yml` 액션이 실행되어 사이트를 배포합니다.

문제가 있거나 다른 배포 경로(예: gh-pages 브랜치)를 원하면 알려주세요.
